Скрипт для парсинга товаров с сайта magbo.ru.
Поддерживает два режима работы:

1. Поиск по запросам

Можно указать один или несколько запросов через запятую.
Скрипт собирает все найденные товары по каждому запросу и парсит их карточки.

2. Прямые ссылки

Можно передать одну или несколько прямых ссылок на страницы товаров.

Установка
pip install -r requirements.txt

Запуск
python main.py
